# PaceUp Run Club — Landing Page

A conversion-focused landing page for **PaceUp Run Club**, a 30-day beginner running program operating across 5 Indian cities. Built as part of a product marketing exercise to demonstrate value communication, funnel structure, and persuasive design.

🔗 **Live site:** `https://rboken22.github.io/Pace-Club/#`

## What's Inside

| File | Purpose |
|------|---------|
| `index.html` | The full landing page — self-contained, no build step needed |
| `.nojekyll` | Tells GitHub Pages to skip Jekyll processing |
| `README.md` | This file |

## Page Structure

The landing page follows a **curiosity → trust → action** flow:

1. **Hero** — Headline, city run cards, and social proof stats
2. **Discover Events** — Live group runs across Delhi, Mumbai, Bengaluru, Pune, Hyderabad
3. **Community** — WhatsApp group preview and join CTA
4. **Is This For You?** — Relatable pain points + member testimonials
5. **30-Day Programme** — Week-by-week breakdown
6. **Gear** — Curated beginner gear with member pricing
7. **Pricing** — Single plan with 7-day free trial and money-back guarantee
8. **FAQ** — Objection handling
9. **Final CTA** — Action close

## Design References

- Visual identity and community language inspired by [Bhag Club](https://www.bhag.co.in)
- Typography: Bebas Neue (display) + Barlow Condensed (labels) + DM Sans (body)
- Color palette: Off-white paper tones with burnt orange accent (`#e04e1a`)

## Target User

Urban Indian adults (20–40) who want to start running but lack structure, confidence, or community to follow through.

## Deploying to GitHub Pages

1. Push all files in this repo to your GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select `Deploy from a branch`
4. Set branch to `main` and folder to `/ (root)`
5. Click **Save** — your site will be live in ~60 seconds at:
   `https://<your-username>.github.io/<your-repo-name>/`
